# GAMEレビュー

## サイト概要
### サイトテーマ
気になっているゲームを買うかどうか基準に使用する、感想を共有するレビューサイト

### テーマを選んだ理由
ゲームが好きなので、そこを基準に考えた時にレビューサイトが思い浮かびました

### ターゲットユーザ
ゲームを買うか悩んでいる人や、ゲームをプレイ済みの人等

### 主な利用シーン
ゲームの感想を共有したい時や、ゲームの内容が知りたい時等
